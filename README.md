# Portafolio Web de Modelados 3D

## Descripción

Este proyecto consiste en un portafolio web desarrollado para mostrar distintos modelos 3D realizados utilizando Blender, ZBrush y Adobe Substance 3D Painter.

La página permite visualizar imágenes, descripciones y videos de los modelos creados, además de ofrecer una sección de contacto e interacciones desarrolladas con JavaScript para mejorar la experiencia del usuario.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Git y GitHub

## Funcionalidades implementadas

### Estructura y diseño
- Diseño completo utilizando HTML y CSS.
- Header con presentación del portfolio y herramientas utilizadas.
- Menú de navegación mediante anclas.
- Secciones de personajes, barcos y videos.
- Footer con información de contacto.
- Diseño responsive para dispositivos móviles y tablets.
- Personalización visual mediante variables CSS.

### Interactividad con JavaScript
- Validación del formulario de contacto.
- Simulación de envío de mensajes.
- Manejo de errores utilizando `try...catch`.
- Buscador dinámico de modelos mediante filtrado de contenido.
- Uso de arrays para almacenar información de los modelos.
- Botón que muestra todos los modelos almacenados en el array.
- Implementación de eventos:
  - `submit`
  - `input`
  - `click`
  - `change`
- Implementación de modo claro mediante un checkbox.
- Manipulación dinámica del DOM para mostrar, ocultar y modificar elementos de la página.

## Características principales

- Visualización de modelos 3D.
- Descripciones detalladas de cada modelo.
- Videos de producción y animaciones.
- Formulario de contacto interactivo.
- Buscador de personajes.
- Modo claro / oscuro.
- Lista dinámica de modelos disponibles.

## Control de versiones

El proyecto fue desarrollado utilizando Git y GitHub para el control de versiones, registrando cada mejora mediante commits descriptivos.
